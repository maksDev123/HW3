# HW3 Databases

## Domain: Work and Travel
## How to run application
### Under Windows
```
python -m venv venv
./venv/Scripts/activate
pip install -r requirements.txt
python ./app.py
```

### Under Mac/Linux
```
python -m venv venv
source ./venv/bin/activate
pip install -r requirements.txt
python ./app.py
```
